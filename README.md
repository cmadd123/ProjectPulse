# ProjectPulse

**Beautiful project communication for contractors and clients**

## Overview

ProjectPulse solves the #1 pain point in home improvement projects: lack of communication and transparency. Contractors post daily photo updates with captions. Clients see a beautiful timeline of their project's progress. Change orders handled in-app. No more endless text messages.

**App Type:** Single app with role-based UI (contractor vs client)  
**Target Market:** 1.1M contractors in US (general contractors, remodelers, handymen)  
**Revenue Model:** B2B SaaS - $25/month Pro, $50/month Premium  
**Build Time:** 4 weeks MVP with Claude Code

---

## Development Commands

```bash
# Navigate to project
cd conscious_apps/projectpulse

# Get dependencies
flutter pub get

# Run on device/emulator
flutter run

# Build release APK
flutter build apk --release
```
